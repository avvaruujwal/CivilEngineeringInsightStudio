# Civil Engineering Insight Studio

Civil Engineering Insight Studio is an AI-powered web application that analyzes construction images and generates engineering insights and landmark descriptions.

## Features
- Construction image analysis
- Safety and material insights
- Landmark/project description generation
- Web interface using Streamlit

## Technologies Used
- Python
- Streamlit
- Pillow
- Generative AI (simulation)

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run application:
   streamlit run app.py
