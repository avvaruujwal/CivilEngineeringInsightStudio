Civil Engineering Insight Studio is an AI-powered web application that analyzes construction images and generates engineering insights along with landmark/project descriptions. This project was developed as part of the SmartInternz Internship Program.
---
##  Project Overview
The application allows users to upload construction site images and receive automated insights such as project type, materials observed, construction stage, safety suggestions, and improvement recommendations. It also generates descriptions for civil engineering landmarks and projects.
